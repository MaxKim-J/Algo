# JS로 알고리즘 풀기 칱싵

## 1. Swap

구조분해 할당을 사용해 swap을 편하게 할 수 있다.

```js
const arr = [1, 2, 3, 4, 5];

[arr[1], arr[2]] = [arr[2], arr[1]];
```

## 2. 변수 여러개 동시에 선언

쉼표를 사용하면 선언자를 한번만 써도 된다

```js
const a = 5, b = 2, c = 3, d = {}, e = [];
```

이것도 구조분해할당의 한 유형인거 같은데, 이렇게도 된다

```js
const [a,b,c,d] = [0,1,2,3]
```
